Estructura de una pagina HTML

Una página HTML es un documento que utiliza el Lenguaje de Marcado de Hipertexto (HTML, por sus siglas en inglés) para mostrar contenido en la web. HTML es un estándar utilizado para crear páginas web y definir su estructura y presentación.


La estructura básica de una página HTML se compone de varios elementos que organizan y presentan el contenido de manera efectiva. Aquí te muestro un esquema básico:



<html>

<head>

<title>Titulo de la pagina</title>

</head>

<body>


</body>

</html>

Etiqueta <p>


Es una de las etiquetas más comunes y esenciales para estructurar texto en una página web. Esta etiqueta se utiliza para definir un párrafo en HTML.


Usos

Se utiliza para agrupar texto que forma una idea o concepto dentro de un documento web.


Estructura

Un párrafo se representa de manera visual como un bloque de texto separado por un margen, lo que ayuda a mejorar la legibilidad. Se escribe como <p>Texto</p>.

Etiqueta <img>


Se utiliza para insertar imágenes en documentos HTML. Es un componente esencial para la creación de contenido visual enriquecido en páginas web, que ayuda a ilustrar temas, mejorar la estética y atraer la atención del usuario.


Usos

Se usa para mostrar imágenes en el contenido de la página.


Estructura

Es una etiqueta auto-cerrada, lo que significa que no necesita una etiqueta de cierre. Se escribe como <img>.

Etiqueta <a>


Se utiliza para crear hipervínculos, uno de los elementos más esenciales de la web. Permite enlazar a otras páginas web, dentro del mismo documento, o a recursos externos.


Usos

  • Enlaces a otras páginas web.
  • Enlaces internos.
  • Abrir enlaces en una nueva pestaña.
  • Enlaces de correo electrónico.

Estructura

Se escribe como <a href="URL_del_destino">Texto del enlace</a>.
